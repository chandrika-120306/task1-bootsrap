About This Project

This project was developed using HTML5, CSS3, and Bootstrap 5. Bootstrap’s grid system and utility classes were used to build a responsive and visually clean layout.

Various Bootstrap components such as navbars, cards, and forms were explored and combined creatively rather than copying full example pages. This helped in understanding component behavior and layout composition.

Focus was placed on spacing, alignment, and consistency to improve overall user experience while maintaining a modern design approach.

GitHub Desktop was used for version control, and the project was deployed using GitHub Pages to provide a live working preview.

Overall, this project improved practical knowledge of responsive UI development and real-world frontend workflows.
